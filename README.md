# Music Collection Analysis

This project analyzes a personal music collection to uncover patterns in artists, release years, and record labels.

## Objective

Explore the dataset and identify trends in:

* Artist frequency
* Release years
* Label distribution

## Tools Used

* Python
* pandas
* seaborn
* matplotlib

## Analysis Includes

* Top artists by number of records
* Distribution of records over time
* Most common labels
* Records grouped by century

## Key Insights

* Pink Floyd is the most represented artist
* The 1970s and 2010s are the strongest decades
* The collection is evenly distributed between 20th and 21st century
